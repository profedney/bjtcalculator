    Metodologia e Fórmulas

    Os cálculos seguem o método clássico de polarização estável:

    • VRE = 0,1 * VCC
    • VRC = 0,4 * VCC
    • VCE = 0,5 * VCC
    • RE = VRE / IE
    • RC = VRC / IE
    • VB = VRE + 0,65V
    • IRB = 0,1 * IE
    • RB1 = (VCC - VB) / IRB
    • RB2 = VB / IRB
